# Лабораторная работа №1
Реализация следующих методов для функции двух аргументов:
1. метод градиентного спуска с постоянным шагом;
2. метод одномерного поиска (метод золотого сечения) и градиентный спуск на его основе;
3. метод одномерного поиска (метод бисекции) и градиентный спуск на его основе.

Тест на сравнение эффективности реализованных методов + метода Нелдера-Мида из библиотеки scipy.optimize по следущим параметрам: сходимость, значение минимума функции, аргументы, количество итераций, количество вычислений градиента, количество вычислений функции.
